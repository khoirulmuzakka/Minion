CEC Benchmark Problems Code (from Suganthan GitHub Repository)
--------------------------------------------------------------

This project includes implementations of the CEC benchmark problems that are based on 
the original code released by [P. N. Suganthan](https://github.com/P-N-Suganthan). The original
code was distributed prior to the CEC competitions for research and benchmarking purposes and, 
to the best of our knowledge, is free to use and redistribute for non-commercial, 
academic, and research applications.
